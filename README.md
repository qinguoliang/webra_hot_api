

# webra_hot
热门网站的热榜

# github发版的版本参考如下方法
```shell
# 将下载下来的webra_top.zip 进行解压缩
unzip webra_top.zip
cd webra_top
chmod +x init
./init
Usage: ./init.sh {start|stop|restart|status}
# 启动
./init start
# 关闭
./init stop
# 重启
./init restart
# 查看状态
./init status
```
